# Roadmunk

> 제품·기술 로드맵 전문 시각화 및 이해관계자 커뮤니케이션 도구

## Overview

Roadmunk는 타임라인 기반의 로드맵 시각화에 특화된 도구다. CTO가 기술 로드맵을 경영진·이사회·팀에 커뮤니케이션할 때, 세부 이슈 관리 도구([[Jira]])와 달리 전략적 수준의 뷰를 제공한다.

## Key Features

- **Timeline View** — 분기별 이니셔티브를 간트 차트 형태로 시각화
- **Swimlane View** — 팀·테마별로 레인을 나눠 병렬 계획 표현
- **Jira 연동** — [[Jira]] 에픽을 가져와 Roadmunk에서 로드맵화
- **다중 뷰 관리** — 이해관계자별(경영진·팀·고객) 다른 뷰 생성
- **공유 기능** — URL 공유 또는 PNG/PDF 내보내기

## CTO 활용 시나리오

| 용도 | 방법 |
|---|---|
| 이사회 보고 | 분기별 기술 로드맵 Timeline 뷰 → PDF 내보내기 |
| 경영진 정렬 | 비즈니스 테마 기반 Swimlane (보안·확장성·신기능) |
| 팀 로드맵 공유 | 팀별 세부 뷰를 별도 생성해 URL 공유 |
| [[OKR]] 연계 | OKR Objective를 로드맵 테마로 매핑 |

## Tips & Shortcuts

- **Now/Next/Later** — 정확한 날짜 없이 우선순위 흐름만 표현하는 경량 방식
- **Milestone 마커** — 주요 배포일·데드라인을 마일스톤으로 표시
- 로드맵은 확정된 계획이 아니라 "현재 의도"임을 이해관계자에게 명확히 전달
- [[Miro]]와 비교: Miro는 자유 형식 워크숍, Roadmunk는 정형화된 로드맵 보고에 적합
