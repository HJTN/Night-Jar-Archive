# Miro

> 온라인 화이트보드 기반 시각적 협업 도구

## Overview

Miro는 무한 캔버스 기반의 온라인 화이트보드로, 원격·하이브리드 팀의 브레인스토밍·워크숍·다이어그램 작업을 지원한다. CTO는 기술 전략 워크숍, 시스템 아키텍처 초안, [[Tech-Radar]] 시각화, 조직 설계 다이어그램에 활용한다.

## Key Features

- **무한 캔버스** — 포스트잇·텍스트·도형·이미지·커넥터를 자유 배치
- **실시간 협업** — 여러 명이 동시에 같은 보드 편집. 커서 위치 실시간 표시
- **Templates** — Tech Radar, Wardley Map, 아키텍처 다이어그램, 회고 등 수백 개 제공
- **타이머·투표** — 워크숍 퍼실리테이션 기능 내장
- **Embed** — [[Confluence]]·[[Notion]]·[[Jira]]에 보드 임베드

## CTO 활용 시나리오

| 용도 | 방법 |
|---|---|
| 기술 전략 워크숍 | 포스트잇+그룹핑으로 팀 아이디어 수렴 |
| [[Tech-Radar]] 시각화 | Radar 템플릿으로 4 Quadrant 배치 |
| 아키텍처 다이어그램 | 시스템 컴포넌트·데이터 흐름 초안 스케치 |
| 조직 설계 | 팀 구조·보고 체계 시각화 |
| 레트로스펙티브 | Start/Stop/Continue 회고 템플릿 활용 |

## Tips & Shortcuts

- `Space` + 드래그 — 캔버스 이동
- `Ctrl+D` — 요소 복제
- **Frames** — 프레임으로 슬라이드처럼 발표 가능
- **Smart Drawing** — 손으로 그린 도형을 자동으로 정형화
- 워크숍 전에 템플릿을 미리 준비해두면 참가자 온보딩 시간 절약
