# Notion

> 문서·데이터베이스·위키·프로젝트 관리를 하나로 통합한 협업 도구

## Overview

Notion은 페이지(문서)와 데이터베이스(테이블·칸반·캘린더)를 자유롭게 조합할 수 있는 올인원 협업 도구다. CTO는 기술 전략 문서, 팀 위키, [[OKR]] 트래킹, 채용 파이프라인 관리 등에 활용한다.

## Key Features

- **Page + Block 구조** — 텍스트·표·코드·이미지·임베드를 블록 단위로 자유 조합
- **Database Views** — 동일 데이터를 Table·Kanban·Calendar·Gallery·Timeline으로 전환
- **Relations & Rollups** — 데이터베이스 간 관계 연결 및 집계 (예: OKR ↔ 프로젝트 연결)
- **AI 기능** — 문서 초안 작성, 요약, 번역 지원
- **Team Wiki** — 구조화된 사이드바로 전사 지식 체계 관리

## CTO 활용 시나리오

| 용도 | 방법 |
|---|---|
| 기술 비전 문서 | Page 작성 후 팀 공유 |
| [[OKR]] 트래킹 | Database로 Objective·KR·달성률 관리 |
| 채용 파이프라인 | Kanban 뷰로 후보자 단계 추적 → [[Engineering-Hiring-Process]] |
| 회의록 | 템플릿으로 반복 구조화, 액션 아이템 자동 추적 |
| 기술 로드맵 | Timeline 뷰로 분기별 이니셔티브 시각화 |

## Tips & Shortcuts

- `[[` — 내부 페이지 링크 (Obsidian과 동일한 위키링크 방식)
- `/template` — 팀 표준 템플릿 적용
- `@` — 팀원 멘션 및 날짜 지정
- **Synced Blocks** — 여러 페이지에서 동일 블록 공유 (업데이트 자동 동기화)
- [[Confluence]]와 비교: Notion은 유연성, Confluence는 Jira 연동이 강점

## Related Tools

- [[Confluence]] — 기업용 위키가 필요하거나 [[Jira]] 연동이 중요한 경우 대안
- [[Jira]] — 이슈 트래킹은 Jira가 더 강력. Notion과 병행 사용
