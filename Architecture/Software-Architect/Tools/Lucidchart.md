# Lucidchart

> 클라우드 기반 다이어그램 협업 도구

## Overview

클라우드 기반 다이어그램·시각화 플랫폼. 실시간 협업과 다양한 템플릿이 강점이며, 기업 팀이 아키텍처 문서를 공동 작업할 때 많이 사용한다.

## Key Features

| 기능 | 설명 |
|---|---|
| **실시간 협업** | 여러 사람이 동시에 편집 가능 |
| **C4 Model 템플릿** | C4 다이어그램 템플릿 제공 |
| **클라우드 아이콘** | AWS / GCP / Azure 공식 아이콘 내장 |
| **Confluence 통합** | Confluence 페이지에 실시간 임베드 |
| **Jira 통합** | Jira 이슈와 다이어그램 링크 연결 |
| **데이터 연동** | CSV/JSON으로 다이어그램 자동 생성 |

## vs Draw.io 비교

| 항목 | Lucidchart | [[Draw-io]] |
|---|---|---|
| **비용** | 유료 (팀 플랜 $9~) | 무료 |
| **협업** | 실시간 협업 강력 | 기본 수준 |
| **템플릿** | 풍부 | 기본 제공 |
| **기업 SSO** | 지원 | 제한적 |
| **Git 통합** | 제한적 | .drawio XML 가능 |

## Common Usage

- 팀 단위 실시간 아키텍처 설계 협업
- 클라우드 아키텍처 다이어그램 작성 (AWS/GCP/Azure 아이콘)
- C4 다이어그램 공동 작성
- Confluence와 긴밀하게 통합된 환경

## Tips & Shortcuts

- `V` — 선택 모드
- `H` — 이동 모드
- `C` — 연결선 모드
- `Ctrl + G` — 그룹 생성
- `Ctrl + Shift + E` — 컨테이너 추가
