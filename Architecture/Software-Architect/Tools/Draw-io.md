# Draw.io (diagrams.net)

> 무료 오픈소스 다이어그램·플로우차트 도구

## Overview

무료 오픈소스 다이어그램 도구. C4 다이어그램, 시스템 플로우, ER 다이어그램 등 아키텍처 문서화에 필요한 모든 다이어그램을 지원한다. 로컬 저장 또는 Google Drive / GitHub 연동 가능.

## Key Features

| 기능 | 설명 |
|---|---|
| **무료·오픈소스** | 라이선스 비용 없음, 셀프 호스팅 가능 |
| **C4 Model 지원** | C4 Shape Library 내장 |
| **VS Code 플러그인** | 코드 저장소 내에서 .drawio 파일 직접 편집 |
| **Git 연동** | .drawio 파일을 XML로 저장 → Git 버전 관리 가능 |
| **Confluence 플러그인** | Confluence에 다이어그램 직접 임베드 |
| **다양한 다이어그램** | 플로우차트, UML, 네트워크 다이어그램, ER |

## Common Usage (아키텍처 작업)

1. C4 Context Diagram — 시스템 전체 경계
2. Container Diagram — 서비스·DB·큐 구조
3. Sequence Diagram — 주요 API 흐름
4. Deployment Diagram — 클라우드 인프라 구성

## Tips & Shortcuts

- `Ctrl + Shift + F` — 도형 검색
- `Ctrl + E` — 선택 요소 스타일 편집
- `Ctrl + Shift + H` / `V` — 수평/수직 정렬
- `F2` — 선택 도형 텍스트 편집 모드
- `Ctrl + Shift + -` / `+` — 줌 조절
