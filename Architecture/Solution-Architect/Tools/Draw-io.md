# Draw.io (diagrams.net)

> 무료 오픈소스 다이어그램·플로우차트 도구

## Overview

무료 오픈소스 다이어그램 도구. 솔루션 아키텍트는 AWS/GCP/Azure 아이콘 라이브러리와 C4 모델 지원을 활용해 클라우드 아키텍처 다이어그램과 솔루션 아키텍처 다이어그램을 작성한다.

## Key Features

| 기능 | 설명 |
|---|---|
| **무료·오픈소스** | 라이선스 비용 없음 |
| **AWS/GCP/Azure 아이콘** | 클라우드 공식 아이콘 내장 |
| **C4 Model 지원** | C4 Shape Library 내장 |
| **VS Code 플러그인** | 코드 저장소 내 .drawio 파일 편집 |
| **Git 연동** | .drawio XML 형태로 Git 버전 관리 |
| **Confluence 플러그인** | Confluence에 다이어그램 임베드 |

## Solution Architect 주요 사용 시나리오

| 다이어그램 | 목적 |
|---|---|
| **솔루션 아키텍처 다이어그램** | 전체 시스템 구조 (AWS 아이콘 활용) |
| **네트워크 다이어그램** | VPC, 서브넷, 방화벽, 연결 구조 |
| **데이터 흐름 다이어그램** | 데이터 입수·처리·저장 흐름 |
| **DR 아키텍처 다이어그램** | 재해 복구 구조 (Multi-Region) |

## AWS 아이콘 사용 방법

1. Draw.io 실행
2. `Extras > Edit Diagram` 또는 왼쪽 패널에서 `+More Shapes`
3. `Networking`, `AWS` 등 Shape Library 활성화
4. 아이콘 드래그앤드롭으로 다이어그램 작성

## Tips & Shortcuts

- `Ctrl + Shift + F` — 도형 검색
- `Ctrl + E` — 선택 요소 스타일 편집
- `Ctrl + Shift + H` / `V` — 수평/수직 정렬
- `Ctrl + Shift + X` — XML 편집 모드 (템플릿 붙여넣기)
