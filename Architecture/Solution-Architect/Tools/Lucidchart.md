# Lucidchart

> 클라우드 기반 다이어그램 협업 도구

## Overview

클라우드 기반 다이어그램·시각화 플랫폼. 솔루션 아키텍트에게는 고객과의 실시간 협업 다이어그램 작성, [[RFP]] 제안서용 전문적인 다이어그램 작성에 강점이 있다.

## Key Features

| 기능 | 설명 |
|---|---|
| **실시간 협업** | 여러 사람이 동시에 편집·댓글 가능 |
| **AWS/GCP/Azure 아이콘** | 클라우드 공식 아이콘 내장 |
| **C4 Model 템플릿** | C4 다이어그램 템플릿 제공 |
| **Confluence 통합** | Confluence 페이지에 실시간 임베드 |
| **기업 SSO** | SAML SSO 지원 |
| **데이터 연동** | CSV/JSON으로 다이어그램 자동 생성 |

## Solution Architect 주요 사용 시나리오

- RFP 제안서용 전문적인 클라우드 아키텍처 다이어그램
- 고객과 실시간 협업으로 아키텍처 초안 작성
- Confluence에 임베드해 SAD(솔루션 아키텍처 문서)와 통합

## Common Usage

1. 고객과 Discovery 세션 중 실시간으로 아키텍처 스케치
2. AWS 기반 솔루션 아키텍처 다이어그램 (공식 아이콘 활용)
3. RFP 응답 다이어그램 작성 (전문적인 시각화)

## Tips & Shortcuts

- `V` — 선택 모드
- `H` — 이동 모드
- `C` — 연결선 모드
- `Ctrl + G` — 그룹 생성
- `Ctrl + D` — 선택 요소 복제
