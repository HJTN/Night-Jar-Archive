# Microsoft Visio

> 다이어그램·프로세스 차트 작성 도구

## Overview

Microsoft의 전문 다이어그램 작성 도구. 기업 환경에서 특히 Microsoft 제품군(Office 365)을 사용하는 조직에서 많이 사용된다. 솔루션 아키텍트가 기업 고객에게 제안서를 작성할 때 Visio 포맷(.vsdx)을 요구하는 경우가 있다.

## Key Features

| 기능 | 설명 |
|---|---|
| **다양한 Shape 라이브러리** | 네트워크, 소프트웨어, 프로세스, 조직도 등 |
| **Azure 아이콘** | Microsoft Azure 공식 아이콘 지원 |
| **SharePoint/Teams 통합** | Microsoft 365 환경과 자연스러운 통합 |
| **Visio Online** | 브라우저 기반 뷰어·편집 (M365 구독 포함) |
| **데이터 연동** | Excel 데이터로 다이어그램 자동 생성 |

## 언제 사용하는가

- 고객이 Microsoft 365 환경을 사용하고 Visio 파일을 요구할 때
- Azure 기반 인프라 다이어그램 (Azure 아이콘 활용)
- 기업 네트워크·인프라 다이어그램 작성
- 프로세스 흐름도·조직도 작성

## Common Usage (Solution Architect)

1. 고객 제출용 솔루션 아키텍처 다이어그램 (.vsdx)
2. Azure 클라우드 아키텍처 다이어그램
3. 기업 네트워크 토폴로지 다이어그램

## Tips & Shortcuts

- `Ctrl + Shift + K` — 연결선 자동 정렬
- `Ctrl + D` — 선택 도형 복제
- `F2` — 선택 도형 텍스트 편집
- Shape 오른쪽 클릭 → `Edit Shape Data` — 메타데이터 입력
