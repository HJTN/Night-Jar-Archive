## 설치 방법

### 1. JTAG/SWD (개발 환경)

```bash
# OpenOCD + ST-Link 사용
openocd -f interface/stlink.cfg -f target/stm32f4x.cfg \
  -c "program firmware_v1.2.3.bin 0x08000000 verify reset exit"
```

### 2. STM32CubeIDE

1. `Run > Run Configurations > STM32 C/C++ Application` 설정
2. 빌드된 `.elf` 파일 선택 후 실행

### 3. OTA 업데이트 (현장 배포)

1. `firmware_v1.2.3.bin` 파일을 OTA 서버에 업로드
2. 디바이스에서 업데이트 명령 전송
3. 부트로더가 자동 다운로드 및 적용
4. 재부팅 후 버전 확인
