# 🗃️ Night-Jar-Archive

> **SW 개발자가 알아야 할 지식을 직무(Job Role) 단위로 체계화한 지식 저장소**

도메인 지식 · 업계 용어 · 주요 업무 프로세스 · 활용 도구를 직무별로 정리했습니다.  
자신의 직무에 맞는 지식을 빠르게 탐색하고, 커리어를 넓혀가는 데 활용해 보세요. 🚀

---

## 📁 저장소 구조

모든 직무 폴더는 동일한 하위 구조를 가집니다.

```
[Job-Role]/
├── 📄 README.md                  # 직무 개요
├── 🧠 Domain-Knowledge/
│   ├── Concepts/                 # 핵심 개념 (1개념 = 1파일)
│   └── Principles/               # 설계 원칙 및 방법론
├── 📖 Terminology/               # 용어 사전 (1용어 = 1파일)
├── ✅ Core-Tasks/
│   ├── Workflows/                # 주요 업무 프로세스
│   └── Checklists/               # 반복 업무 체크리스트
├── 🛠️ Tools/                     # 직무별 SW 도구 정리
└── 💼 Projects/
    └── Templates/                # 프로젝트 문서 템플릿
```

## 🔍 활용 방법

### 직무 탐색하기
1. 위 직무 목록에서 관심 직무를 찾는다
2. 해당 폴더의 `README.md`로 직무 개요를 파악한다
3. `Terminology/`에서 모르는 용어를 검색한다
4. `Core-Tasks/Workflows/`에서 실제 업무 흐름을 익힌다
5. `Tools/`에서 직무에서 사용하는 도구를 학습한다

### GitHub에서 검색하기
- 저장소 내 전체 파일 검색: 이 페이지에서 `.` 키를 누르면 VS Code 웹 에디터가 열립니다
- 특정 키워드 검색: 아래 형식을 GitHub 검색창에 입력하세요

```
repo:HJTN/Night-Jar-Archive 검색어
```

### 💜 Obsidian과 함께 사용하기
이 저장소를 클론해 Obsidian의 Vault로 열면 노트 간 **링크 연결**과 **Graph View**를 바로 활용할 수 있습니다.

```bash
git clone https://github.com/HJTN/Night-Jar-Archive.git
```
